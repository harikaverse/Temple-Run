# Temple Run Using Hand Gestures 🎮🖐️
A computer vision-based game control system that allows users to play Temple Run using hand gestures instead of a keyboard.


## 🚀 Features
-  Control the game using hand gestures
-  Real-time motion detection
-  Interactive gameplay experience
-  Works with webcam


## ⚙️ Technologies Used
- Python 
- OpenCV 
- MediaPipe 
- PyAutoGUI (for controls)


## 🧠 How it Works
The system detects hand landmarks using MediaPipe and tracks finger movements.  
Different gestures are mapped to game controls such as:
- 👉 Move Right
- 👈 Move Left
